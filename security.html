🔐 AuthX Workshop — OAuth2 & OIDC Playground

Goal: Explore and understand OAuth2 and OpenID Connect using Keycloak as an Identity Provider. This hands-on project sets up a complete environment to test client authentication flows, token issuance, and policy enforcement via OPA.

  • 🧪 Provision demo clients, realms, roles, and users via Terraform.
  • ⚙️ Run a Keycloak server using Docker Compose with preconfigured clients and admin UI.
  • 🔄 Test password grants, client credentials, and token introspection.
  • 🛡️ Integrate Open Policy Agent (OPA) for fine-grained authorization.
  • 📂 Includes Makefile automation for local testing and sandbox provisioning.

🧠 Ideal for learning how modern identity, access management, and policy enforcement works in real-world applications.

🔐 OPA Workshop — Policy-Driven Microservices

Goal: Build a modular environment to test Open Policy Agent integration with modern microservices.

  • ✅ Supports local, bundle server, and S3-based Rego policy loading
  • 🔒 Enforces field-level access with JWT and OAuth2 authorization
  • 🧪 Demonstrates real-world authorization workflows using Sanic API
  • 🚀 Deployable via Docker and Helm with K3d simulating Kubernetes

🧠 Ideal for understanding how policy enforcement works in cloud-native apps.
